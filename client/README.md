# The Craving - Restaurant Landing Page

A modern fast food restaurant landing page built with React and Tailwind CSS.

## Features
- Fully responsive design
- Smooth scroll animations with Framer Motion
- Order delivery and table booking form
- Dark theme with custom color palette

## Tech Stack
- React
- Tailwind CSS
- Framer Motion
